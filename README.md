# ChargeNode DevOps

Mobile-friendly PWA for interacting with Azure DevOps on the go — view boards, trigger pipelines and manage releases from your phone.

## Tech stack

- React + TypeScript + Vite
- PWA with service worker and offline support
- Hosted on GitHub Pages

## Getting started

```bash
npm install
npm run dev
```

## Build & deploy

```bash
npm run build
```

The `dist/` folder is deployed to GitHub Pages with base path `/ChargeNodeDevOps/`.
