# CLAUDE.md

## Project overview

ChargeNode DevOps — a PWA (Progressive Web App) for Azure DevOps, built with React + TypeScript + Vite. Targets mobile (Android primarily, iOS secondary). Hosted on GitHub Pages.

## Key features (planned)

1. Auth — PAT token or OAuth against Azure DevOps
2. Boards — View/move/create work items
3. Pipelines — List, trigger, view status
4. Releases — Trigger releases

## Commands

- `npm run dev` — start dev server
- `npm run build` — TypeScript check + production build
- `npm run lint` — ESLint
- `npm run preview` — preview production build locally

## Conventions

- Language: TypeScript (strict)
- Styling: plain CSS (no framework yet)
- Base path: `/ChargeNodeDevOps/` (GitHub Pages)
- Git commits: no AI attribution lines
